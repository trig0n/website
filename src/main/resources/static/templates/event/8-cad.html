cad tools i've used - an overview

  • openscad
  • solidpython
  • solvespace
  • tinkercad
  • freecad

openscad

text based
model is viewable
a few model designs are enough to learn most features
lots of code exports to stl, off, amf, dxf, svg, csg and png

official site, github


solidpython

text based
model needs to be viewed in openscad
needs to be compiled twice (python & opencad)
less code than openscad
scad file doesn't need to be reloaded after recompiling python code
as easy to learn as openscad

github


solvespace

drawing based
not able to make real threads
first tutorial and some curiosity teaches you basically everything you need to know
no code
features/shortcuts can be learned in less than a day
exports to a heck lot of formats for a heck lot of different purposes

official site, github


tinkercad

drawing based
can be used without doing the tutorial
no code
exports to obj, stl and svg
also is able to send the model to your printer

official site


freecad

drawing based
able to make threads
gotta check the documentation when creating your first few models
no code
exports to a heck lot of formats
