eclipse

different versions for different languages
not my favourite
code completion is not the best


codeblocks

a little worse than eclipse


pycharm

good code completion
version control integration
code formatting and a lot of other stuff
has a free community edition witch fulfills all hobby programmer needs


idea

can be used for basically any language (mainly using it for java though)
profiling and coverage
android sdk integration
version control integration
cant suggest using it for android projects (android studio does it better)


clion

c / c++ ide
profiling / coverage
version control
cmake based


arduino

no code completion
indentation helper
board / library
need to load a json via settings to support certain boards


atom

better code completion than arduino ide
supports a lot of boards
highly customizable


android-studio

code completion
android sdk integration
version control
coverage / profiling
(pretty good for developing apps)
