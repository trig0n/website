experience

piplay works without any problems with the exception for pi_server.exe
pi_server.exe crashes sometimes, especially when the device isn't connected or gets disconnected
the correct nvidia driver actually make a difference (fixes bluescreens / low frame rate)
render quality found in piplay actually affects the image quality, but apparently does not affect the frame rate
supplied usb cable for head sensor is too long. tried another cable of the same length didn't work either (usb 3.0 / 5V, 2A)
3m usb cable is good enough
hdmi cable is not removable/exchangeable


usage

place base station at shoulder level or if higher slightly tilted downwards
stand ~2m away from the base station
remove reflective objects facing the base station
reduce light, radio- and microwaves


development
